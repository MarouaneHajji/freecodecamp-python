# Apply Discount Function

A small Python project that calculates the final price of an item after applying a percentage discount, with input validation.

Features:
    - Validates that price and discount are numbers (int or float).
    - Ensures price is greater than 0.
    - Ensures discount is between 0 and 100.
    - Calculates the final price after applying a valid discount.
    - Returns error messages for invalid input.
